# 欧易OKX交易所注册账号及身份认证教程

以下是注册欧易交易所账号的具体操作流程。

# **一、欧易账户注册**

1. 点击下方欧易交易平台注册链接，进入欧易注册页面。  
   **[https://www.okx.com](https://www.okx.com/join/74873351)**

2. 输入邮箱地址，支持国内163邮箱、QQ邮箱或海外Outlook邮箱、Gmail邮箱等。  
   使用注册链接时，邀请码将自动显示；如未显示，请手动输入邀请码 **74873351**。  
   使用此邀请码可在后续交易中永久享受20%手续费优惠。  
   点击【注册】按钮。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1ee74fe18d_1709541470-60.jpg)

3. 输入邮箱验证码，点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8297c1802e_1709541786-61.jpg)

4. 输入区号+86（中国）的手机号码，点击【立即验证】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/651f965f7f_1709541949-62.jpg)

5. 输入发送到手机上的6位数验证码，点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a6e7bd31ec_1709542032-63.jpg)

6. 在居住国家/地区选项中，选择“中国”，点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/85ab8ebd8b_1698380134-5.jpg)

7. 设置密码，要求至少8个字符，包含1个大写字母、1个小写字母、1个数字和1个标点符号。完成后点击【下一步】，完成账户注册。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/048705e710_1698380596-6.jpg)

# **二、下载欧易APP**

1) 安卓手机下载  
   安卓用户可通过浏览器访问国内镜像网址下载安装APP。  
   下载过程中如出现病毒提示，请直接忽略。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/fab004487d_2024030409030126.jpg)

2) 苹果手机下载  
   1. 苹果手机用户需先注册外区Apple ID，可参考下方视频操作；如不成功或嫌麻烦，可在百度购买外区Apple ID账号（仅需几元）。  
   2. 获取海外Apple ID后，打开App Store，点击右上角头像，滑动至底部，点击【登出】退出国内ID。  
      ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/07c84de9ce_1709535360-47.jpg)  
   3. 输入注册好的海外Apple ID登录。  
      ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4579185041_1709005874-10.jpg)  
   4. 在搜索栏输入“欧易”，选择下方欧易应用下载安装。  
      ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/19c2fa5ebf_1709612035-8.jpg)

# **三、欧易实名认证**

1. 下载完成后打开欧易APP，点击【立即认证】进行实名认证，提升账户安全。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b1880ef77e_1709544122-20.jpg)

2. 选择居住国家/地区为“中国”，证件类型为“身份证”，点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2127ec2d3f_1709544254-21.jpg)

3. 仔细阅读证件上传要求，准备好身份证件，点击【开始】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8b15d41167_1709544355-22.jpg)

4. 按要求拍摄身份证正面（人像面），点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/abd3336592_1709544392-23.jpg)  
   再拍摄背面（国徽面），点击【下一步】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/16cc48265f_1709544412-24.jpg)

5. 输入姓名，选择证件类型“身份证”，填写身份证号，点击【下一步】。根据提示完成人脸扫描即可认证。  
   注意：必须填写真实身份信息，否则人脸识别无法通过。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e6d1f1c31f_1709545548-25.jpg)

# **四、账户安全设置**

为加强账户安全，需进行以下安全设置。

1. 进入欧易首页，点击左上角点阵，选择“个人资料和设置”，点击【安全设置】，再选【安全中心】。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7a1cd36fe0_1710940893-19.jpg)

2. 绑定“身份验证应用”、“手机号”和“邮箱”。这三项均建议设置。  
   ![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3c847ca6c5_1710941027-20.jpg)  
   注册时已绑定手机号和邮箱。  
   身份验证器需按提示下载谷歌身份验证器（下载方法与欧易APP相同）。  
   绑定后，登录或提现等操作需输入账户密码及谷歌验证码，实现双重验证保障资金安全。
