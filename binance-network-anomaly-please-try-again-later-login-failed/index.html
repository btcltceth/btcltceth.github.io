【币安网络异常请稍后再试】币安登录不了？

很多朋友反映自己的币安账号无法登录。本文就来为你解决币安账号登录不上的问题，教你如何保持网络通畅，随时顺利登录币安。

一、币安APP下载

如果你还没下载币安APP，强烈建议你先下载一个币安APP。

  1. 安卓手机点击下方国内镜像网站，拉到底部下载APK：
    https://vlink.cc/okxchina

  2. 苹果用户需要先有一个台湾区的Apple ID，可以查看下方视频，按流程注册Apple ID。

  3. 注册完成后，打开App Store，点击右上角的小人头像，滑到最下方，点击【登出】，退出原来的中国大陆Apple ID。

  4. 滑到最上方，点击登录按钮，使用你的台湾区Apple ID登录。搜索“binance”，找到第二个就是币安APP，点击打开。

二、币安账户注册

  1. 由于币安的网络需要通过多个服务器中转，有时会出现卡顿。但正常使用时，不需要开启梯子或VPN工具，就能正常访问。

    点击左上角的币安图标，再点击【设置】按钮。

  2. 如果还是无法登录，就在设置页面往下滑，找到【清理缓存】，将其清理掉。

  3. 清理完成后，关闭币安，重新打开币安APP，它会重新加载。点击左上角的【注册或登录】。

  4. 点击左下角的“创建一个币安账户”，再点击【通过邮箱或手机号码注册】。

  5. 输入你的邮箱或手机号进行注册。创建好币安账号后，才能填写邀请码。输入邀请码36457687，即可永久获得20%的手续费优惠。记得点击邀请码后面的【提交】进行绑定，才能享受优惠。

三、币安购买USDT

  1. 点击右上角的【充值/买币】，选择【快捷买币】。

  2. 上方可以选择快捷区或自选区。右上角的CNY就是我们要使用的法币（人民币），可以切换，这里我们使用人民币CNY。

  3. 选择【买入】，币种选择USDT。USDT是泰达币，目前币圈使用最广，类似于美元。用法定货币购买USDT后，就可以用USDT兑换任意数字货币。

    筛选金额，比如想买500元的USDT，输入500，点击【确认】。

  4. 接着筛选支付方式，有支付宝、银行卡和微信支付，选择你习惯的方式。

  5. 币安会筛选出商家。选择成单量和成单率都较高的商家，比如第二家，成交量9102，好评率100%，算是很高的。点击【买入】。

  6. 再次输入购买金额，系统会自动计算。支付500元人民币，将收到约69.63颗USDT。点击【买入USDT】。

  7. 订单生成。仔细阅读广告方的交易条款，他要求买家必须实名付款，且与平台名字一致，否则不放币。阅读后，点击【去付款】。

  8. 按商家提供的支付方式转账，完成后点击【我已付款，通知卖家】，让他放币。

  9. 购买的USDT会存放在资金账户。点击右下角的【资金】，选择【资金账户】，即可看到购买的USDT。

四、币安交易

  1. 点击【划转】，从资金账户划转到现货账户。币种选择USDT，全部划转，点击【确认划转】，转账成功。

  2. 划转到现货账户后才能交易。点击下方的【交易】按钮，选择【现货】，再选择交易币种。比如看好比特币，就搜索BTC（比特币代码），选择第一个BTC/USDT，即用USDT兑换比特币。想兑换其他币种，也可在此搜索并用USDT兑换。

  3. 在交易页面中，红色的是卖出价格，绿色的是买入价格，中间的数字是实时成交价，即比特币的报价。

  4. 选择【买入】，主要下单方式有【限价单】和【市价单】。

  5. 限价单：自己报价，价格接近时才能成交。比如当前价格43355，想以43300买入，就在价格处输入43300，再输入数量或金额，点击【买入BTC】。

  6. 下单成功，但未成交，只是请求成功。可在下方当前委托看到挂单，需跌到指定价格才能成交。如果不想等了，点击后面的【撤单】撤销请求。挂单和撤单若不成交，不收取手续费。

  7. 如果想立即买入，选择【市价单】。它会以当前最优价格直接成交，输入购买金额即可。点击【买入BTC】。

  8. 下单成功，比特币已购买成功。

    点击右下角的【资金】，选择【现货账户】，即可看到购买的比特币。
